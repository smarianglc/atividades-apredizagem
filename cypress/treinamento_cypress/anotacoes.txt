----------PASSOS PARA CRIAR UM PROJETO--------------
---------------comandos no terminal-----------------
1. no terminal dentro da pasta execute o comando 
    - npm init --yes (é criado um arquivo package.json para com informações de scripts que será executado)

2. instalar o pacote Cypress
    - npm install -D cypress

3. criar uma estrutura de pastas padrão
    - npx cypress open (responsavel por abrir o cypress e criar, pode demorar um tempinho e dar erro mas tente novamente)

------criando pastas quando abre o cypress---------
- fixtures (para armazedas dadose arquivos de mocks)
- integration (paraarmazenar arquivos de testes)
- plugsins (configuração de plugs)
- support (arquivos auxiliarese de suporte ao teste)

- PASTAS CRIADAS DURANTE OS TESTES:
- screenshots (printd/screenshots durate  teste)
- videos (fazer videos durante os testes)
